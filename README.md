# Rules, Policies, and Practices

## The HMC Honor Code applies

Harvey Mudd College is an inclusive community of faculty, staff and students. 
Students entering the community are assumed to have an earnest purpose. Diligent 
pursuit of this purpose is enhanced by standards of conduct agreed upon by the 
community. These standards are:

1. Thoughtful respect for the rights of others;
2. Honesty and integrity in both academic and personal matters;
3. Responsible behavior both on and off campus;
4. Appropriate use of campus buildings and equipment, and;
5. Compliance with College regulations and policies

## What the Honor Code Means for this GitHub Organization

Any project shared in this organization should comply with the honor code. 

## The use of AI

AI generated code can be used in this organization. The ideal practice is to 
always share when code is generated by AI. Approved/Recommended AI models are:
- Antropic Claude (known for creating safe and aligned AI models)
- Google DeepMind (focuses on human-centered AI, rejecting the use of AI for weapons or surveillance)

When using AI models, please remember that all educational information is 
protected under FERPA and that you should not be uploading or sharing 
educational information that you may have access to such as
- academic transcripts, grades, and GPA
- (personal student) class schedules and attendance records
- disciplinary records, special education records, and advising notes
- student health/medical records
- financial aid records
